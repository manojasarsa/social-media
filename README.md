# ALCON - Always Connected - Social Media App

ALCON is a social media app made with `HTML`, `Tailwind CSS`, `JavaScript`, `Reactjs` and `redux-toolkit`.

## Features

In ALCON, you can:
### `Create a Post`

- Like Post
- Comment & Replies

### `Edit & Delete Post`
### `User Feed`

- Sort by Date, Trending & Recent
### `User Profile`

- Add Profile Picture
- Add Bio
- Add Portfolio URL
### `Follow/UnFollow`

### `Explore Feed`

### `Bookmark Post`

### `Authentication`

- Sign-up
- Login
- Logout
